# 🕷️ Spiderman 2D Adventure Game

A simple 2D side-scrolling Spiderman game built using HTML, CSS, and JavaScript.

## 🎮 Game Description -
In this game, you control Spiderman and complete the journey within a given time limit. 
You must defeat enemies (aliens) using web attacks while surviving with only 2 lifelines.

## 🚀 Features -
- 🕸️ Web shooting attack
- 👾 Enemy (alien) system
- ❤️ 2 life system
- ⏱️ Time-based gameplay
- 🏆 Score and kill counter
- 🎨 Animated UI and background

## 🕹️ How to Play -
- Click **Play** to start the game
- Use controls to move and attack enemies
- Kill enemies to increase score
- Avoid damage — you only have 2 lives
- Finish the journey before time runs out
